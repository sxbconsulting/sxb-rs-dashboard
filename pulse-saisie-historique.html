Guide — Où trouver tes stats ?

Pour chaque plateforme, le chemin exact à suivre sur ton téléphone ou ordinateur pour récupérer chaque métrique à saisir dans l'app.

📸
Instagram
Prérequis : compte Professionnel ou Créateur activé
📱 Application mobile
1
Active le compte Professionnel si pas encore fait
⚙️ Paramètres → Compte → Passer en compte professionnel → Créateur
Gratuit. Te donne accès à toutes les statistiques.
2
Ouvre ton Tableau de bord professionnel
Ton profil → Bouton "Tableau de bord professionnel" (sous ta bio)
S'il n'apparaît pas, va dans ⚙️ Paramètres → Outils créateur → Tableau de bord.
3
Accède aux statistiques complètes
Tableau de bord → "Voir les informations" ou "Statistiques"
4
Sélectionne la période
En haut → sélecteur de période → "7 derniers jours"
Où trouver chaque chiffre
Abonnés total
→
Ton profil → chiffre sous "Abonnés"
Nouveaux abonnés
→
Statistiques → onglet "Audience" → "Abonnés gagnés"
Portée hebdo
→
Statistiques → onglet "Aperçu" → "Comptes touchés"
Impressions
→
Statistiques → onglet "Aperçu" → "Impressions"
Taux d'engagement
→
Calcul : Interactions ÷ Portée × 100 (ex : 320 ÷ 6100 × 100 = 5.2%)
Clics lien bio
→
Statistiques → "Activité" → "Clics sur le lien du site"
Leads générés
→
Manuel : compte tes DM entrants qualifiés de la semaine
💡 Astuce : Fais une capture d'écran de l'écran "Aperçu" chaque lundi avant de saisir. Tu auras tout sous les yeux.
💼
LinkedIn
Prérequis : aucun — tout profil standard a accès aux analytiques
💻 Web ou mobile
1
Va sur ton profil
Clique sur ta photo → "Voir le profil"
2
Ouvre tes analytiques
Sur ton profil → section "Analytiques" → clique sur n'importe quelle stat pour tout voir
Sur mobile : fais défiler sous ta photo de profil.
3
Accède aux stats de tes posts
Analytiques → "Impressions des publications" → période : 7 jours
Où trouver chaque chiffre
Connexions / abonnés
→
Ton profil → "Connexions" (en haut)
Nouveaux abonnés
→
Analytiques → "Abonnés" → variation sur 7 jours
Impressions hebdo
→
Analytiques → "Impressions des publications" → 7 derniers jours
Taux d'engagement
→
Calcul manuel : (Likes + Commentaires + Partages) ÷ Impressions × 100
Commentaires
→
Analytiques → "Interactions" → Commentaires (7 jours)
Partages
→
Analytiques → "Interactions" → Republications (7 jours)
Leads générés
→
Manuel : compte tes DM entrants qualifiés de la semaine
💡 Astuce LinkedIn : note aussi tes 2-3 meilleurs posts de la semaine (titre + impressions). Ça t'aide à identifier tes formats gagnants dans le temps.
🎵
TikTok
Prérequis : compte Créateur ou Business (gratuit à activer)
📱 Application mobile
1
Active le compte Créateur si pas encore fait
☰ Menu → Paramètres → Compte → "Passer en compte créateur"
Gratuit. Nécessaire pour accéder aux analytics détaillés.
2
Accède aux Outils créateur
Ton profil → ☰ (coin haut droit) → "Outils créateur"
3
Ouvre Analytics
Outils créateur → "Analytics" → onglet "Vue d'ensemble"
4
Sélectionne la bonne période
En haut → "7 derniers jours"
Où trouver chaque chiffre
Abonnés total
→
Ton profil → chiffre "Abonnés"
Nouveaux abonnés
→
Analytics → Vue d'ensemble → "Abonnés" (variation +/-)
Vues hebdo
→
Analytics → Vue d'ensemble → "Vues vidéo"
Taux de complétion
→
Analytics → onglet "Contenu" → clique une vidéo → "Temps de lecture moyen" ÷ durée × 100
Taux d'engagement
→
Calcul : (Likes + Commentaires + Partages) ÷ Vues × 100
Meilleure vidéo
→
Analytics → onglet "Contenu" → vidéo en haut = la plus vue cette semaine
Leads générés
→
Manuel : DM qualifiés + clics sur lien bio
💡 Astuce TikTok : le taux de complétion est l'indicateur le plus important pour l'algorithme. Si tu es sous 40%, tes vidéos sont trop longues ou l'accroche n'est pas assez forte.
▶️
YouTube
Prérequis : avoir une chaîne YouTube liée à ton compte Google
💻 studio.youtube.com
1
Va sur YouTube Studio
Navigateur → studio.youtube.com → connecte-toi avec ton compte Google
Sur mobile : app "YouTube Studio" (à télécharger séparément).
2
Ouvre les Analytiques
Menu gauche → "Analytiques"
3
Sélectionne la période
En haut à droite → "7 derniers jours"
Où trouver chaque chiffre
Abonnés total
→
Studio → Résumé (page d'accueil) → "Abonnés actuels"
Nouveaux abonnés
→
Analytiques → onglet "Audience" → "Abonnés gagnés"
Vues hebdo
→
Analytiques → onglet "Aperçu" → "Vues"
Heures de visionnage
→
Analytiques → onglet "Aperçu" → "Durée de visionnage (heures)"
Rétention moyenne
→
Analytiques → onglet "Engagement" → "Durée de visionnage moyenne (%)"
Clics CTA
→
Analytiques → onglet "Engagement" → "Clics sur les fiches / écrans de fin"
Leads générés
→
Manuel : clics lien description + DM / commentaires qualifiés
💡 Astuce YouTube : la rétention moyenne est l'indicateur roi. Au-dessus de 50% = excellente vidéo. En dessous de 35% = revois ton intro (les 30 premières secondes font tout).
🗓️ Ton rituel du lundi — 20 min chrono
Suis ces étapes dans l'ordre chaque lundi matin pour une saisie sans friction
☐Ouvre Instagram → Tableau de bord pro → capture d'écran des stats 7j
☐Ouvre LinkedIn → Analytiques → capture d'écran impressions + abonnés
☐Ouvre TikTok → Outils créateur → Analytics → capture vue d'ensemble
☐Ouvre YouTube Studio → Analytiques → capture aperçu 7 jours
☐Compte tes DM qualifiés de la semaine sur chaque plateforme
☐Saisie hebdo → entre les chiffres plateforme par plateforme
☐Clique "Sauvegarder"
☐Clique "Exporter CSV" → dépose dans Google Drive
